Техническая документация к программе "3D Viewer"

1. Введение

Данный документ представляет подробное описание функционала и инструкции по работе с программой "3D Viewer". Эта программа предназначена для визуализации каркасной модели в трехмерном пространстве, разработана на языке Си стандарта C11 с использованием компилятора gcc и дополнительных библиотек и модулей QT.

2. Основные функциональные возможности

Программа предоставляет следующие функциональные возможности:

  • Загрузка каркасной модели из файла формата obj (поддержка только списка вершин и поверхностей).
  • Перемещение модели на заданное расстояние относительно осей X, Y, Z.
  • Поворачивание модели на заданный угол относительно своих осей X, Y, Z.
  • Масштабирование модели на заданное значение.
  • Графический пользовательский интерфейс для визуализации и управления моделью.
  • Сохранение настроек между перезапусками программы.
  • Сохранение отрендеренных изображений в форматах bmp и jpeg, запись "скринкастов" в gif-анимацию.

3. Графический пользовательский интерфейс

Интерфейс программы включает следующие элементы:

  • Загрузка каркасной модели из файла формата obj (поддержка только списка вершин и поверхностей).
  • Перемещение модели на заданное расстояние относительно осей X, Y, Z.
  • Поворачивание модели на заданный угол относительно своих осей X, Y, Z.
  • Масштабирование модели на заданное значение.
  • Визуализация модели в графическом пользовательском интерфейсе.
  • Информация о загруженной модели - название файла, количество вершин и рёбер.

4. Установка и настройка

Код программы размещается в папке src. Сборка программы настроена с помощью Makefile со стандартным набором целей для GNU-программ: all, install, uninstall, clean, dvi, dist, tests, gcov_report. Установка программы может быть выполнена в любой другой произвольный каталог с помощью команды `make install`. Для запуска программы нужно использовать команду `make open`

5. Дополнительные настройки

Программа предоставляет возможность настройки типа проекции, типа и цвета рёбер, способа отображения вершин, а также сохранения настроек между перезапусками программы.

6. Запись

Программа позволяет сохранять отрендеренные изображения в форматах bmp и jpeg, а также записывать аффинные преобразования загруженного объекта в gif-анимацию.

Заключение

"3D Viewer" - это эффективный инструмент для визуализации каркасных моделей в трехмерном пространстве. Следуя инструкциям из данной документации, пользователи смогут полноценно использовать все возможности программы.
